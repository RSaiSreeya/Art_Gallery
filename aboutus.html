The Face Behind This Project

Hello! I am Sai Sreeya Ramasahayam, a student currently pursuing the Application Development for Analytics course at the University of South Florida.

This website was designed and developed as part of my learning journey. Tasked with building a static website based on a content theme, I chose to create an Art Gallery platform that blends simplicity, elegance, and functionality.

The development process began with careful planning of the layout and purpose of each page, from creating an inviting homepage, to showcasing a profile, simulating data operations, and visualizing art-related trends. I used HTML to build the foundation, CSS to define the aesthetic, and selected a calm beige theme to mirror the ambiance of an art exhibit.

For interactivity, I used JavaScript. For analytics, I integrated Chart.js. The project was built using Visual Studio Code, versioned via GitHub, and tested live with Live Server, making development iterative and effective.

This experience sharpened my front-end skills and deepened my appreciation for thoughtful design. It represents a meaningful step in my path as a developer, one I look forward to expanding upon.
